Notas git

INICIAR REPOSITORIO EN LA CARPETA ACTUAL

 git init

CLONAR REPOSITORIO

 git clone

DEFINIR REMOTE

 git remote add origin https://github.com/mrgitman/holamundo.git

AÃ‘ADIR FICHEROS AL STAGING INDEX

PARA VER EL ESTADO DEL STAGING INDEX

ACTUALIZAR CAMBIOS PARA SUBIR

 git commit -am "comentario"

CREAR RAMA

 git branch desarrollo

CAMBIAR DE RAMA

 git checkout desarrollo

VER RAMAS

 git branch

SUBIR AL SERVIDOR (CAMBIOS O RAMAS)

 git push

 git push -u origin master

 git push -u origin desarrollo

MERGE AL MASTER DESDE OTRA RAMA

 git checkout master
 git merge desarrollo

BORRAR RAMA

 git branch -d desarrollo

DESCARGAR CAMBIOS DESDE EL SERVIDOR

 git pull

DESCARTAR CAMBIOS LOCALES Y DESCARGAR TODO DESDE EL SERVIDOR

 git fetch origin
 git reset --hard origin/master
