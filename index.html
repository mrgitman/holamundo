Notas Git

init

Crear un nuevo repositorio en la carpeta actual

 git init

clone

Descargar un repositorio ya existente sobre la carpeta actual

 git clone

remote

Repositorios remotos

 git remote add origin https://github.com/mrgitman/holamundo.git

 git remote -v

add

Mueve al índice las modificaciones que hayamos hecho

 git add fichero

 git add --all

status

Ver el estado de las modificaciones en el índice

 git status

commit

Ejecuta los cambios añadidos con add

 git commit -am "comentario"

branch

Ver, crear y borrar ramas

 git branch

 git branch nombre-rama

 git branch -d nombre-rama

checkout

Cambiar de rama

 git checkout nombre-rama

push

Envía los cambios al servidor

 git push

 git push -u origin master

 git push -u origin nombre-rama

merge

Combinar cambios al master desde otra rama

 git checkout master
 git merge nombre-rama

pull

Descargar cambios desde el servidor

 git pull

fetch y reset

Descartar cambios locales y descargar todo otra vez desde el servidor

 git fetch origin
 git reset --hard origin/master
